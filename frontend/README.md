# Carpool Frontend (React + Vite + Leaflet)

- Зависимости: `npm install`
- Стартиране: `npm run dev` (порт 5173)
- API проксира се към `http://localhost:8080` за `/api`

Картата използва Leaflet + OpenStreetMap. Компонентът `RideMap` показва маршрут (от/до град) и точки за качване; може да се включи режим за избор на място при клик.
